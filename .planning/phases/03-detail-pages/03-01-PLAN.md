```typescript
function generateSlug(tradeName: string): string {
  return tradeName
    .toLowerCase()
    .replace(/[^a-z0-9\s-]/g, '')
    .replace(/\s+/g, '-')
    .replace(/-+/g, '-')
    .replace(/^-|-$/g, '');
}
```
The current slug function does NOT strip legal suffixes. Collisions are handled by appending a numeric suffix (e.g., `-2`), not by town name.
