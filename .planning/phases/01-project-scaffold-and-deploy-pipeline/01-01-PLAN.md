  <action>
    Create the Astro project in the current project root (C:/Users/mmcge/MACannabisProject) -- NOT in a subdirectory. The .planning/ directory and .xlsx files already exist here.

    **Step 1: Initialize Astro 5 project**
    Run `npm create astro@5 -- --template minimal --no-install` in a temp directory, then copy the generated files into the project root (to avoid overwriting .planning/ and other existing files). Alternatively, manually create the project files. After setup, run `npm install` and then `npm install tailwindcss @tailwindcss/vite`.

    Verify Astro version is 5.x with `npx astro --version`. If it installed 6.x, downgrade with `npm install astro@5`.

    **Step 2: Configure Astro with Tailwind v4**
    Create `astro.config.mjs`:
    ```javascript
    import { defineConfig } from "astro/config";
    import tailwindcss from "@tailwindcss/vite";

    export default defineConfig({
      site: "https://dispensaries.meredithmcgee.org",
      vite: {
        plugins: [tailwindcss()],
      },
    });
    ```

    Do NOT use `@astrojs/tailwind` (deprecated for v4). Do NOT install `@astrojs/cloudflare` (not needed for static sites). Do NOT add `output: 'static'` (it is the default).

    **Step 3: Create global CSS**
    Create `src/styles/global.css` with just `@import "tailwindcss";`

    **Step 4: Create BaseLayout**
    Create `src/layouts/BaseLayout.astro` with:
    - Import of `../styles/global.css` in frontmatter
    - HTML shell with lang="en", charset, viewport meta
    - `<title>` set to "MA Cannabis Directory" (per user decision)
    - Meta description: "Find out who owns every licensed cannabis dispensary in Massachusetts. A transparent, searchable ownership directory."
    - `<slot />` for page content

    **Step 5: Create placeholder homepage**
    Create `src/pages/index.astro` with:
    - Import and use BaseLayout
    - Centered layout with "Who Owns Your Dispensary?" headline (per user decision)
    - "MA Cannabis Directory -- Coming Soon" subtitle
    - Three stat boxes: 525 Active Licenses, 92% Independently Owned, 157 Towns (per user specifics)
    - Footer: "A project by Meredith McGee" -- subtle, small text (per user decision)
    - Use Tailwind utility classes throughout (green-700 for stat numbers, gray tones for text)

    **Step 6: Create supporting files**
    - `public/robots.txt`: Allow all crawlers, reference future sitemap
    - `public/favicon.svg`: Simple green leaf or cannabis-themed SVG icon
    - `data/.gitkeep`: Empty file to create the data/ directory for Phase 2

    **Step 7: Create .gitignore**
    Must exclude: `node_modules/`, `dist/`, `.astro/`, `.env`, `.env.*`, `*.xlsx`, `*.xls`, `.DS_Store`, `Thumbs.db`, `.vscode/`

    **Step 8: Initialize git and push to GitHub**
    Run `git init` (repo is not yet initialized). Create initial commit with all project files. Create a GitHub repository using `gh repo create` (public or private per user preference -- default to public since it is a public directory site). Push to main branch.

    Important: The .xlsx files in the project root MUST be excluded by .gitignore before the initial commit. Verify with `git status` that no .xlsx files are staged.
