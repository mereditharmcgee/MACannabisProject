  <name>Task 2: Build XLSX parser module with row-level error reporting</name>
  <files>src/lib/xlsx-parser.ts, tests/validation.test.ts, tests/data-pipeline.test.ts</files>
  <behavior>
    - Test: parseDispensarySheet reads headers from row 4 of "Dispensary Directory" sheet
    - Test: parseDispensarySheet extracts data rows starting from row 5
    - Test: parseDispensarySheet skips empty rows (no Trade Name and no License #)
    - Test: parseDispensarySheet returns validated Dispensary objects for valid rows
    - Test: parseDispensarySheet returns errors array with row number and field name for invalid rows
    - Test: error message format includes "Row N (License #XXX): field 'fieldName' ..."
    - Test: error message for row without license includes "Row N (unknown): ..."
    - Test: getCellString handles null, undefined, number, rich text object, and string cell values
    - Test: parseSpecialStatusTags splits comma-separated tags and maps to valid enum values
    - Test: parseSpecialStatusTags ignores unrecognized tag values
    - Test: parseSummarySheet extracts totalLicenses, percentIndependent, totalTowns from Summary sheet
  </behavior>
  <action>
    1. Create `src/lib/xlsx-parser.ts`:
       - Import ExcelJS, path, and dispensarySchema from schemas
       - getCellString(cellValue: ExcelJS.CellValue): string | null -- handles all ExcelJS cell types:
         - null/undefined -> null
         - string -> trimmed string (empty string -> null)
         - number -> String(value)
         - object with 'richText' property -> concatenate text parts
         - Date -> toISOString()
         - other -> String(value)
       - parseSpecialStatusTags(raw: string | null): string[] -- splits on comma/semicolon, trims each, filters to valid specialStatusTags enum values, ignores unrecognized
       - parseDispensarySheet(workbook: ExcelJS.Workbook): { records: Dispensary[], errors: string[], warnings: string[] }
         - Get worksheet by name "Dispensary Directory" (throw if not found)
         - Read headers from row 4
         - Map header strings to schema field names (create a headerMap: { 'Trade Name': 'tradeName', 'License #': 'licenseNumber', 'Owner': 'owner', ... })
         - Iterate rows from 5 onward
         - Skip rows where Trade Name AND License # are both empty
         - Transform raw row data to schema shape (using headerMap + getCellString)
         - Parse Special Status column into specialStatusTags array
         - Set needsNarrative = true if ownershipDetails is null/empty
         - Set researchInconclusive = true if owner matches /research inconclusive/i
         - Generate slug from tradeName (lowercase, replace spaces/special chars with hyphens, deduplicate)
         - Validate through dispensarySchema.safeParse()
         - On success: add to records
         - On failure: format error with row number, license # (or "unknown"), and Zod issue paths/messages, add to errors
         - Track warnings for: empty phone, empty address, empty owner (but not build-failing)
       - parseSummarySheet(workbook: ExcelJS.Workbook): Stats
         - Get worksheet by name "Summary"
         - Extract numeric stats by scanning for labeled cells (e.g., cell containing "Active Licenses" -> read adjacent/below cell for number)
         - Return { totalLicenses, percentIndependent, totalTowns } with sensible defaults if not found

    2. Create `tests/validation.test.ts`:
       - Import parseDispensarySheet from xlsx-parser
       - Use ExcelJS to create test workbooks programmatically in beforeEach
       - Tests per behavior list above for error reporting
       - Test that a workbook with a row missing Trade Name produces an error for that row

       - Test: parseSummarySheet returns stats object
